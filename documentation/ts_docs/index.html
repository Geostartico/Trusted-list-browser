Trusted list server browser

Application written in Typescript and React, whose purpose is to show the user a list of trusted servers based on the parameters selected by the user with a web-based fancy GUI.

Getting Started

Prerequisites

For all os and distros: to use this application you need STABLE INTERNET CONNECTION

Windows

  • Node.js

    1. Download node (LTS recommend) from this page
    2. Run node -v command on terminal to verify node installed correctly

  • yarn

    Download yarn from here

    Run following code on powershell to verify installation

    yarn --version

Linux
Debian/Ubuntu

  • Node.js

    curl -fsSL https://deb.nodesource.com/setup_18.x && sudo apt install -y nodejs

    Also try command

    node --version

    to verify Node installed correctly

  • npm 

    sudo apt install npm

  • typescript

    npm install -g typescript

  • yarn

    npm install --g yarn

MacOS

  • Brew

      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

  • Node.js and npm

      brew install node

    Also try command

    node --version
    npm --version

    to verify Node installed correctly

  • yarn

    npm install -g yarn

    Installation

  1. Open terminal and move to Trusted-list-browser/src/trusted_list/ folder

  2. run command

    yarn install

    Usage

  3. Move to Trusted-list-browser/src/trusted_list/ folder, then start program using yarn start command. Now application should start automatically on your default browser. If not, open your browser manually then type in the address bar localhost And you should be connected to the server.

  4. From GUI you can select and deselect items from the checkbox list that appears and application will show you a server trusted list based on your previous filter selection
